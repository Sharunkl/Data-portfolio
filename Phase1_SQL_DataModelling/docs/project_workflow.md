## Overview

This project is to analyze the library dataset within a residental complex. 

## Step 1 : Data Points Identification

### High-level hierarchy flow 

The major aspects of this project are as below
```text
Residential Complex -> Blocks -> Sector -> Projects -> Towers -> Flats -> Residents [Taken with reference with Parsn apartments in coimbatore]
Library -> Collections [Books , Magazines , NewsPaper ]  -> Memberships -> Book_Issue_Register
```
Residential Complex is the community living space where this library is located
Projects are different community projects implemented within this space
